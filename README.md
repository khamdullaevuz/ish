## Task

### Installation

#### Install composer packages
```shell
composer install
```
#### Create .env file
```shell
cp .env.example .env
```
#### Generate
```shell
php artisan key:generate
```

#### Migrate
```shell
php artisan migrate
```

#### Seed
```shell
php artisan db:seed
```

#### Run
```shell
php artisan serve
```
